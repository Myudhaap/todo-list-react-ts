### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

- Text Editor
- NodeJS
- PNPM

### Installation

1. Clone the repo
         https://github.com/Myudhaap/todo-list-react-ts
   
2. Install package dependencies

        pnpm install
   

3. Run the app

        pnpm vite
